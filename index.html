Welcome to Mr. Roboger's Neighborhood!

Here in the neighborhood, we like to count! But we do it a little different because we like to say silly words and make new friends!

We like to replace some of our numbers with silly words! Here's how it goes:

  • Any number that contains a 1, we call Beep!
  • Any number that contains a 2, we call Boop!
  • Any number that contains a 3, we like to ask that number, Won't you be neighbor?

Keep in mind:

  • Any number that contains 3 will be asked to join the neighborhood.
  • Any number that contains 2 but NOT 3, will be called Boop!

Let's get started! Enter a number below and you'll get a list of all your beeps and boops and all the new friends you've invited to the neighborhood.
